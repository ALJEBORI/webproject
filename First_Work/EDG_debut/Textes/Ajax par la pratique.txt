Présentation par l'éditeur
Ajax s'est imposé comme technique privilégiée de programmation d'interfaces depuis l'essor des applications orientées Web, et particulièrement Web 2.0. Le code côté client redevient un élément fondamental de l'interaction avec les services Web, et les bibliothèques JavaScript facilitant les interactions entre les utilisateurs et les serveurs de données fleurissent pour toutes les plates-formes.

Ajax par la pratique se veut aussi complet que possible sur les différents moyens de mettre en oeuvre Ajax, indépendamment du navigateur, et quelle que soit l'architecture utilisée. Vous apprendrez bien sûr à maîtriser le comportement de l'objet XMLHttpRequest sur toutes les plates-formes, mais vous vous familiariserez surtout avec les bibliothèques et APIs les plus intéressantes comme Script.aculo.us, Prototype, Dojo, ou E4X.
